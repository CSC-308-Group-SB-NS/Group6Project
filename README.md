Car Dealership! - Purpose is to show the user a list of car brands, allow them to select one and choose a car in that list. Then they can look at some details about the cars and purchase them accordingly. 

Group Members: Nicco Settimi and Sam Boston 

List of Corse concepts used 
  1. Sending data between view controllers 
  2. Segues
  3. Constraints/Auto layout
  4. Tableview
  5. Extension
  6. Functions
  7. Outlets
  8. Navigation Controller
  9. Optionals 
  10. Animations
  11. Images

New Concepts Used
  1. UICollectionView - Used to make a set of clickable brands the user can choose from, as well as displaying the number of cars from that brand. 
     References: https://developer.apple.com/documentation/uikit/uicollectionview
     https://medium.com/@coolanil.saini/ios-uicollectionview-a-complete-guide-3152746763cc

Testing: All you need to do to use this app is select what you want to choose and continue from there, nothing too out of the ordinary.
